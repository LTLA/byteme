Detailed Description

template<typename Type_ = char, class Pointer_ = Reader*>
struct byteme::PerByteParallel< Type_, Pointer_ >

Perform parallelized byte-by-byte extraction from a Reader source.

Template Parameters
Type_Character type to return, usually char for text or unsigned char for binary.
Pointer_A (possibly smart) non-const pointer to a Reader.

This is much like PerByte except that the Reader's loading operation is called in a separate thread, thus allowing the caller to parse the bytes of the current chunk in parallel.
