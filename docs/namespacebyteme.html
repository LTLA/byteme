template<typename Size_ >
std::size_t byteme::cap ( Size_ size)
constexpr

Cap an integer at the largest value that can be represented by a std::size_t. This avoids silent integer overflows, especially in the defaults of the various *Options classes.

Template Parameters
Size_Integer type, typically for some buffer size.
Parameters
sizeNon-negative integer specifying some kind of buffer size.
Returns
size if it fits in a std::size_t, otherwise the largest value of a std::size_t.
